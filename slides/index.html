Loops

Loops


var animals = [ 'Cat', 'Dog', 'Owl' ];

// Loop over array using conventional for loop
for( var i = 0; i < animals.length; i++ ) {
  if( animals[ i ] === 'Dog' ) {
    console.log( 'Who let the dog in here?' );
  }
}

// Loop over array using forEach
animals.forEach( function( e ) {
  if( e == 'Dog' ) {
    console.log( 'Who let the dog in here?' );
  }
});

What is the DOM?

Document Object Model

"The Document Object Model is an API for HTML documents. It provides a structural representation of the document, enabling the developer to modify its content and visual presentation. Essentially, it connects web pages to scripts or programming languages."

Some Examples…

Get a reference to an element object


// Get element(s) by className
var logo = document.getElementsByClassName( 'logo' );

// Get element(s) by tagName
var divs = document.getElementsByTagName( 'div' );

// Get element by ID
var container = document.getElementById( 'container' );

// Query Selector[All]
var matches = document.querySelectorAll( 'h2.logo, #container' );

Event Binding


// Get a reference to the `.logo` element object
var logo = document.querySelector( '.logo' );

// Bind a `click` event to the logo
logo.addEventListener( 'click', function( event ) {
  event.preventDefault(); // prevent element's default action
  console.log( 'you clicked me!' );
});

What is jQuery?

"jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript."

Some Examples…

Get a reference to an element object


// Get element(s) by className
var logo = $( '.logo' );

// Get element(s) by tagName
var divs = $( 'div' );

// Get element by ID
var container = $( '#container' );

// Get multiple items
var matches = $( 'h2.logo, #container' );

Event Binding


// Get a reference to the `.logo` element object
var logo = $( '.logo' );

// Bind a `click` event to the logo
logo.click( function( event ) {
  event.preventDefault(); // prevent element's default action
  console.log( 'you clicked me!' );
});

What is JSON?

JavaScript Object Notation, is a text-based open standard designed for human-readable data interchange. It is derived from the JavaScript scripting language for representing simple data structures and associative arrays, called objects. Despite its relationship to JavaScript, it is language-independent, with parsers available for many languages.
